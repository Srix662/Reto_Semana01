# Calculadora de Sumas (Reto Semana 01)

La funcion principal de este programa es leer líneas de texto con datos separados por comas, limpia caracteres inválidos, trunca decimales a enteros y calcula la suma total por línea, ignorando espacios y líneas vacías.

## Instrucciones de Uso

Para ejecutar este programa, necesitas tener Python instalado. Puedes pasarle los datos a través de la entrada estándar (stdin) leyendo un archivo de texto.

Ejecuta el siguiente comando en tu terminal:
python main.py < tests/entrada.txt > tests/salida.txt

## Ejemplo de Entrada y Salida

**Archivo de entrada:**
1,2,3
10

1.9,2.1,3.7
1a2,3b,4

**Salida esperada en consola:**
6
10
0
6
19
